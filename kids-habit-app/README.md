# 儿童习惯养成微信小程序 🌟

一个专为儿童设计的习惯养成小程序，通过游戏化的方式帮助孩子建立良好的生活习惯。

## ✨ 功能特点

### 🏠 首页（任务打卡）
- 显示孩子头像、昵称、当前积分
- 今日任务完成进度展示
- 连续打卡天数统计（🔥 连续 X 天）
- 好习惯任务（绿色，完成加积分）
- 坏习惯惩罚（红色，触发减积分）
- 打卡庆祝动画（撒花/星星）
- 积分飘字动画效果
- 支持取消打卡（误操作时积分同步回退）
- 全部任务完成时特别提示

### 🏆 成就页（徽章系统）
- 7 种预设徽章：
  - 🌱 第一次打卡
  - 🔥 连续 3 天
  - ⭐ 连续 7 天
  - 💎 连续 30 天
  - 🎯 单日全部完成
  - 🛒 首次兑换
  - 💰 累计 500 积分
- 已解锁徽章彩色显示，未解锁灰色
- 点击查看徽章详情和解锁条件

### 🎁 商城页（积分兑换）
- 8 种预设奖励（可自定义）
- 积分足够高亮显示，不足显示"差 XX 分"
- 兑换确认弹窗
- 成功动画效果
- 兑换历史记录查看
- 自动解锁"首次兑换"徽章

### ⚙️ 设置页（家长管理）
- 4 位数字密码保护
- 错误 3 次锁定 5 分钟
- 首次使用引导设置密码
- 任务管理：增删改（名称、图标、积分值、类型、频率）
- 奖励管理：增删改（名称、图标、所需积分）
- 孩子档案：设置昵称和头像（表情符号选择）
- 音效开关
- 手动重置今日任务
- 清空所有数据（二次确认）

## 🎨 设计特色

- **暖色调主题**：橙色/黄色/粉色，温馨活泼
- **卡通风格**：适合儿童的视觉设计
- **大字体**：清晰易读，保护视力
- **动画效果**：庆祝动画、积分飘字等增强趣味性
- **表情符号**：使用 emoji 作为图标，无需额外图片资源

## 🛠️ 技术架构

### 前端
- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 微信云开发数据库

### 后端
- 微信云函数（获取 openid）
- 云开发数据库（5 个集合）

### 数据隔离
- 每个家庭数据相互隔离
- 使用微信 openid 作为用户标识
- 所有数据存储在云端，多设备同步

## 🚀 快速开始

### 前置准备

1. 注册微信小程序账号（https://mp.weixin.qq.com）
2. 下载并安装微信开发者工具（https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html）
3. 开通微信云开发（免费额度足够使用）

### 部署步骤

#### 1. 导入项目
- 打开微信开发者工具
- 选择"导入项目"
- 选择 `kids-habit-app` 文件夹
- 填入你的小程序 AppID（测试可用测试号）

#### 2. 配置云开发
- 在 `app.js` 中填入你的云环境 ID：
```javascript
wx.cloud.init({
  env: 'your-env-id', // 替换为你的云环境 ID
  traceUser: true,
})
```

#### 3. 部署云函数
- 在开发者工具中，右键点击 `cloud/login` 文件夹
- 选择"上传并部署：云端安装依赖"

#### 4. 创建数据库集合
在云开发控制台创建以下集合（权限设置为"所有用户可读写"）：

- `users` - 用户数据
- `tasks` - 任务数据
- `rewards` - 奖励数据
- `exchanges` - 兑换记录
- `achievements` - 成就徽章

详细字段说明见 `images/README.md`

#### 5. 准备图标
准备 8 个 tabBar 图标（81x81 像素）放入 `images` 文件夹：
- `home.png`, `home-active.png`
- `achievement.png`, `achievement-active.png`
- `mall.png`, `mall-active.png`
- `settings.png`, `settings-active.png`

可从 iconfont.cn 下载免费图标。

#### 6. 编译运行
- 点击编译按钮
- 在真机或模拟器中测试

## 📝 使用说明

### 家长首次设置

1. 打开小程序，点击底部"设置"标签
2. 点击"家长设置区"卡片
3. 首次使用会提示设置 4 位数字密码（如：1234）
4. 进入家长管理界面

### 添加任务

1. 在设置页点击"+ 添加任务"
2. 填写任务名称（如：按时起床）
3. 选择任务图标
4. 设置积分值（建议 5-20 分）
5. 选择任务类型：
   - ✅ 好习惯（完成加积分）
   - ❌ 坏习惯（触发减积分）
6. 选择频率：每日/每周
7. 点击保存

### 添加奖励

1. 在设置页点击"+ 添加奖励"
2. 填写奖励名称（如：看动画片 30 分钟）
3. 选择奖励图标
4. 设置所需积分（建议 50-300 分）
5. 点击保存

### 设置孩子档案

1. 在设置页"孩子档案"区域
2. 从表情符号中选择喜欢的头像
3. 点击"保存档案"

### 孩子日常使用

1. 打开小程序，查看今日任务
2. 完成任务后，点击任务卡片打卡
3. 观看庆祝动画，获得积分
4. 积分足够后，在商城兑换奖励
5. 收集成就徽章

## 🎯 教育建议

### 任务设计原则

1. **具体明确**：避免"表现好"，使用"按时起床"、"自己刷牙"等具体行为
2. **难度适中**：孩子努力一下就能完成
3. **正向激励**：以好习惯任务为主，坏习惯惩罚为辅
4. **循序渐进**：初期任务少一些，逐步增加

### 积分设置建议

- 简单任务：5-10 分（如：自己穿衣）
- 中等任务：10-15 分（如：按时起床）
- 困难任务：15-20 分（如：主动学习）
- 坏习惯惩罚：5-10 分（不宜过高）

### 奖励设置建议

- 小奖励：50-80 分（如：小零食、贴纸）
- 中奖励：100-150 分（如：看动画片、去公园）
- 大奖励：200-300 分（如：新玩具、小礼物）

## ⚠️ 注意事项

1. **数据安全**：定期在云开发控制台备份数据
2. **密码保管**：家长密码请妥善保管，忘记后需清空数据重置
3. **适度使用**：建议每天使用，避免过度依赖积分激励
4. **亲子互动**：积分只是手段，家长的陪伴和肯定更重要
