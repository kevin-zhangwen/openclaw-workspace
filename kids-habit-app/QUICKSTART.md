# 快速开始指南 🚀

> 零代码基础也能看懂的部署教程

## 第一步：准备工作（10 分钟）

### 1. 注册微信小程序
1. 访问 https://mp.weixin.qq.com
2. 点击"立即注册"
3. 选择"小程序"
4. 填写邮箱、密码等信息
5. 完成邮箱激活
6. 信息登记（个人选择"个人"，企业选择"企业"）
7. 等待审核（通常很快）

### 2. 下载开发工具
1. 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载对应系统的版本（Windows/Mac）
3. 安装微信开发者工具
4. 使用微信扫码登录

### 3. 开通云开发
1. 在微信开发者工具中，点击顶部工具栏的"云开发"按钮
2. 首次使用会提示开通
3. 同意协议，创建环境
4. 记住环境 ID（格式类似：cloud1-xxx）

## 第二步：导入项目（5 分钟）

1. 打开微信开发者工具
2. 点击"+"号或"导入项目"
3. 项目目录：选择 `kids-habit-app` 文件夹
4. AppID：
   - 如果已有小程序：填入你的 AppID
   - 如果没有：选择"测试号"（功能受限但可体验）
5. 点击"导入"

## 第三步：配置云开发（3 分钟）

1. 打开 `app.js` 文件
2. 找到这一行：
```javascript
// env: 'your-env-id', // 填入你的云开发环境 ID
```
3. 将 `your-env-id` 替换为你的云环境 ID
4. 去掉前面的 `//` 注释符号
5. 保存文件（Ctrl+S 或 Cmd+S）

例如：
```javascript
wx.cloud.init({
  env: 'cloud1-abc123', // 你的环境 ID
  traceUser: true,
})
```

## 第四步：上传云函数（2 分钟）

1. 在左侧文件树找到 `cloud/login` 文件夹
2. 右键点击 `login` 文件夹
3. 选择"上传并部署：云端安装依赖"
4. 等待上传完成（看到成功提示）

## 第五步：创建数据库（5 分钟）

1. 点击顶部工具栏的"云开发"按钮
2. 进入云开发控制台
3. 点击左侧"数据库"
4. 点击"+"号创建集合

需要创建 5 个集合：

### 集合 1：users
- 集合名称：`users`
- 权限设置：选择"所有用户可读写"

### 集合 2：tasks
- 集合名称：`tasks`
- 权限设置：选择"所有用户可读写"

### 集合 3：rewards
- 集合名称：`rewards`
- 权限设置：选择"所有用户可读写"

### 集合 4：exchanges
- 集合名称：`exchanges`
- 权限设置：选择"所有用户可读写"

### 集合 5：achievements
- 集合名称：`achievements`
- 权限设置：选择"所有用户可读写"

## 第六步：准备图标（10 分钟）

小程序底部的 4 个标签需要图标，准备 8 个图片文件：

### 方法 1：使用现成图标（推荐）
1. 访问 https://www.iconfont.cn
2. 搜索关键词：home, trophy, shopping, settings
3. 下载 PNG 格式，尺寸 81x81 像素
4. 每个图标下载两个版本（灰色和橙色）
5. 重命名后放入 `images` 文件夹

### 方法 2：使用工具生成
1. 访问 https://icon.kitchen
2. 选择图标和颜色
3. 下载 PNG 格式
4. 重命名后放入 `images` 文件夹

### 需要的文件名：
```
images/
├── home.png              (灰色房子)
├── home-active.png       (橙色房子)
├── achievement.png       (灰色奖杯)
├── achievement-active.png (橙色奖杯)
├── mall.png              (灰色购物袋)
├── mall-active.png       (橙色购物袋)
├── settings.png          (灰色齿轮)
└── settings-active.png   (橙色齿轮)
```

## 第七步：编译运行（1 分钟）

1. 点击开发者工具顶部的"编译"按钮
2. 等待编译完成
3. 在模拟器中查看效果
4. 点击"预览"可在真机测试（需要绑定开发者）

## 第八步：首次使用（5 分钟）

### 在模拟器或真机中：

1. **进入设置页**
   - 点击底部第 4 个标签"设置"

2. **设置家长密码**
   - 点击"家长设置区"卡片
   - 首次使用会提示设置密码
   - 输入 4 位数字（如：1234）
   - 点击确定

3. **设置孩子档案**
   - 在"孩子档案"区域
   - 点击选择一个表情符号作为头像
   - 点击"保存档案"

4. **添加第一个任务**
   - 点击"+ 添加任务"
   - 任务名称：按时起床
   - 选择图标：⏰
   - 积分值：10
   - 类型：✅ 好习惯
   - 频率：每日
   - 点击保存

5. **添加第一个奖励**
   - 点击"+ 添加奖励"
   - 奖励名称：看动画片
   - 选择图标：📺
   - 所需积分：100
   - 点击保存

6. **返回首页打卡**
   - 点击底部第 1 个标签"首页"
   - 看到刚才添加的任务
   - 点击任务卡片打卡
   - 观看庆祝动画！🎉

## ✅ 完成！

现在你已经成功部署并使用了这个小程序！

## 🆘 常见问题

### Q: 导入项目时提示"AppID 无效"
A: 检查是否正确复制了 AppID，或者先使用"测试号"体验功能。

### Q: 云函数上传失败
A: 确认已开通云开发，并且 `app.js` 中配置了正确的环境 ID。

### Q: 数据库创建失败
A: 确认云开发环境已开通，且有足够的配额（免费版足够使用）。

### Q: 底部标签不显示图标
A: 检查 `images` 文件夹中是否有 8 个图标文件，文件名是否正确。

### Q: 点击打卡没反应
A: 检查云函数是否部署成功，数据库集合是否创建。

### Q: 如何绑定真机测试？
A: 
1. 在微信公众平台（mp.weixin.qq.com）
2. 进入"成员管理"
3. 添加你的微信号为"体验成员"
4. 在开发者工具中点击"预览"，扫码即可
