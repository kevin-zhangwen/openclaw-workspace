📦 系统要求与安装

SYSTEM REQUIREMENTS & INSTALLATION

系统要求

操作系统Linux / macOS / Windows
Node.jsv20+
内存≥4GB (推荐 8GB)
存储≥1GB 可用空间
网络WebSocket 支持

快速安装

# 安装 OpenClaw CLI

npm install -g openclaw

# 或从源码安装

git clone https://github.com/openclaw/openclaw

验证安装

  • openclaw --version 查看版本
  • openclaw --help 查看帮助
  • openclaw doctor 健康检查
  • openclaw status 状态查询

⚙️ 配置与认证

CONFIGURATION & AUTHENTICATION

STEP 01 - 初始化配置

运行 openclaw setup 创建 ~/.openclaw/openclaw.json 和工作空间文件

STEP 02 - 模型认证

配置 AI 模型 API Key：
export ANTHROPIC_API_KEY="..."
或使用 openclaw models auth paste-token --provider anthropic

STEP 03 - Channel 配置

连接通讯渠道：
openclaw channels login --verbose
支持 WhatsApp、Telegram、Signal、飞书等

STEP 04 - 启动 Gateway

openclaw gateway 启动网关服务
默认端口：18789 (WebSocket)

⚠️ SECURITY: API Key 存储在 ~/.openclaw/.env 或系统环境变量，确保文件权限安全！

🌐 Gateway 架构

GATEWAY ARCHITECTURE

核心组件

WebSocket Server

  • 端口：18789 (默认)
  • 协议：JSON-RPC 2.0
  • 支持多客户端并发

Session Manager

  • 会话状态管理
  • JSONL 持久化存储
  • 位置：~/.openclaw/agents/<id>/sessions/

Tool Router

  • 工具调用路由
  • 权限控制
  • 结果聚合

配置管理

  • 配置文件：~/.openclaw/openclaw.json
  • 环境变量：~/.openclaw/.env
  • 支持 Profile 隔离：--profile <name>
  • 热重载配置：openclaw config apply

安全机制

  • Exec 审批模式（off/on-miss/always）
  • 工具白名单控制
  • 凭证加密存储
  • 会话隔离

🤖 Agent 运行时

AGENT RUNTIME

Agent Loop 生命周期

📨 消息接收
→
📚 上下文组装
→
🧠 模型推理
→
🔧 工具执行
→
📤 响应流式

工作空间

  • 默认：~/.openclaw/workspace
  • 核心文件：AGENTS.md, SOUL.md, TOOLS.md
  • 记忆系统：MEMORY.md + memory/*.md
  • Sandbox 隔离（可选）

流式输出

  • Assistant deltas 实时流
  • Tool 事件流（start/update/end）
  • Lifecycle 事件（start/end/error）
  • 支持 Block 流式分块

🔧 Skills 机制

SKILLS SYSTEM

Skills 来源

  • Bundled: 内置技能（随安装包）
  • Managed: ~/.openclaw/skills
  • Workspace: <workspace>/skills
  • 优先级：Workspace > Managed > Bundled

Skill 结构

skill-name/

├── SKILL.md

├── index.js

├── package.json

└── assets/

示例 Skills

  • 🌤️ weather - 天气查询
  • 🔍 web_search - 网络搜索
  • 📊 github - GitHub CLI
  • 📝 feishu-doc - 飞书文档
  • 🎨 frontend-slides-cn - PPT 生成

⚡ 性能优化

PERFORMANCE OPTIMIZATION

内存管理

  • 会话历史限制：messageLimit
  • 工具输出截断：maxOutputChars
  • 后台任务 TTL：cleanupMs
  • 定期清理旧会话

  • 会话级序列化（防冲突）
