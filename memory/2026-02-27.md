### 技术栈

- github.com/shirou/gopsutil/v3
- 内置 net/http

### 运行方式
```bash
./mydba -dsn "user:pass@tcp(host:port)/mysql" -port 8080
```

### 注意事项
- 默认 DSN: `root:password@tcp(127.0.0.1:3306)/mysql`
- 默认端口：8080
- 仪表盘每 3 秒自动刷新
- 无 MySQL 连接时仍可运行（仅显示系统指标）
